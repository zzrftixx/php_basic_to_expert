# PHP: (php Hypertext Preprocessor)

Sebuah bahasa pemrograman open source besar yang di buat pada tahun 1995 oleh Ramus 
Lerdorf, dengan menggunakan bahasa server side scripting dan di adobsi untuk 
pembuatan website (Backend) bahasa ini menjadi bahasa paling populer, dan bersaing 
dengan bahasa besar lainnya seperti javascript, python, golang, c++, c dan bahasa 
besar lainnya. Dan faktanya bahasa ini menjadi bahasa yang digunakan sekala besar,
bahkan website di dunia mayoritas menggunakan php.

- [php.net](https://www.php.net/)
- [github php-src](https://www.php.net/https://github.com/php/php-src)
